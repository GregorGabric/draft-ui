### Vertical

<Callout type="warning">
  Unlike **`RadioGroup`**, **`CheckboxGroup`** does not support
  `aria-orientation`. So `data-orientation` is used to style the layout instead.
</Callout>

Set the `orientation` prop to `vertical` to use a vertical **`CheckboxGroup`**.

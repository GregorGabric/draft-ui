### Vertical

Set the `orientation` prop to `vertical` to use a vertical **`Slider`**.

<Callout type="important">
  You must also add `orientation="vertical"` to **`SliderFilledTrack`** and give
  **`SliderTrack`** a specific height.
</Callout>

### Horizontal

<Callout type="warning">
  Unlike **`CheckboxGroup`**, **`RadioGroup`** does support `aria-orientation`.
  So that is used to style the layout.
</Callout>

Set the `orientation` prop to `horizontal` to use a horizontal **`RadioGroup`**.
